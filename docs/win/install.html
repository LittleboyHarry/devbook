关于分区​

Q: 硬盘有必要分区？怎样分区？

A:

在我看来，2022 年已经没有故意分区的必要性，除非装双系统。 对固态硬盘而言，软件同操作系统装在 C 盘上能得到很好的使用体验。 定期使用云盘或装入硬盘盒的机械硬盘备份重要数据更加靠谱。

添加英语键盘​

编程、游戏用途（需要纯英文输入模式），请添加 英语(美国)-美式键盘 作为备用输入法 （ 语言选择时，按 End 键 > 再 3~4 次 PageUp 就能找到 ）

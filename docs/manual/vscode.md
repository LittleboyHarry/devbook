# VSCode 技巧

## 实用命令缩写

按下 `Ctrl + Shift + P` 打开命令行面板：

 <div className="no-table-header no-table-border">

|            |                              |
| ---------- | ---------------------------- |
| joinl      | 合并行                       |
| stages     | git 提交选中行的修改         |
| amend      | git amend                    |
| revertfile | 撤销文件修改                 |
| rpath      | 复制相对文件路径             |
| cpath      | 复制绝对文件路径             |
| tclear     | 清除终端内容                 |
| maxpanel   | 底部面板最大化               |
| closeoe    | 关闭其它标签页               |
| dsr,dsrex  | 分析拖慢起动和运行速度的插件 |

</div>

## 标签太多，如何避免混乱？

- 有新的浏览动机或想法时，按下 `Ctrl + N` 弹出新窗口
- 向下拖拽标签也会产生新窗口
- 多标签管理:
  - Ctrl + 单击 —— 多选
  - Shift + 单击 —— 连选

善于利用浏览器支持多窗口的特性，以避免太多标签挤在同一个窗口。

推荐的快捷键：仿 Windows 风格

<div className="autoselect-cell-of-table no-table-border">

| 作用       | 按键               |
| ---------- | ------------------ |
| 文件管理器 | Super + E          |
| 设置       | Super + I          |
| 系统监视器 | Ctrl + Shift + Esc |
| 截图 \*    | Super + Shift + S  |

\*注：

- F1 打开终端
- GNOME 42 已移除 `gnome-screenshot` 并强化了 `PrtScr` 截图键的功能、支持录屏

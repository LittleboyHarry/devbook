---
title: Sysinternals
---

安装

    winget install sysinternals

| 模块            | 功能                             |
| --------------- | -------------------------------- |
| Autoruns        | 管理自动运行等加载项，优化系统   |
| ProcessExplorer | 更好的任务管理器                 |
| ProcessMonitor  | 进程监控器（开发者调试程序必备） |
| TCPView         | TCP 流量监控                     |

## 推荐工具

必装：`git` `curl` `wget` `python3`

- dos2unix: 与 Windows 换行符格式转换
- tree: 文件树打印
- unzip: 解压
- man: 帮助手册

更多工具在[开发章节内](/docs/dev/modern-cli/index)

## 建议

尽量把文件存储在 WSL 文件系统内：

Windows 文件系统兼容层并不靠谱，具有不能监听文件修改、文件管理器交换文件时产生碎片文件等 BUG

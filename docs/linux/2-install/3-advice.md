## 手动分区

- 保留 Windows MSR 恢复分区：微软建议每个系统分区后都保留着一个，不要删除
- 使用 EFI 分区：Ubiquity, Calamares 称为“引导器”，Ananconda 则挂载到 `/boot/efi`
- 若手动挂载过分区：别忘了在安装系统前先手动卸载

## 各发行版注意事项

### Ubuntu

1. Welcome 面板：选择语言 - 拖到最底部选 `中文(简体)`，然后“试用”
2. 对 Ubuntu 的专业用户而言，推荐选择速度更快的“最小化安装“

其他细节按默认和个人喜好选择即可

截图识字 tesseract

  • scoop
  • choco
  • pacman
scoop install tesseract
sudo choco install tesseract
yes | sudo pacman -S tesseract tesseract-data-chi_sim

默认仅支持英文

for Scoop: 建议下载到 %TESSDATA_PREFIX% 中 ( ... 其他所有语言 )

简体中文识别模型

使用​

Win + Shift + S 截图后必须打开“截图工具”窗口，识别最后一次截图的命令：

tesseract (ls $env:LOCALAPPDATA\Packages\Microsoft.ScreenSketch_8wekyb3d8bbwe\TempState\ | sort LastWriteTime | select -Last 1).FullName stdout -l chi_sim

删除 -l chi_sim 后缀提高纯英文截图的识别效果
